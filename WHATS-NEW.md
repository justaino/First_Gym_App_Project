# What's New — Testing Guide 💪

Thanks for helping test the app! This round adds **per-set planning**, a much
better **workout mode that saves as you go**, and the ability to **edit/delete
past workouts**. Below is each change and simple steps to try it.

## Before you start (read this first)
- Open the app link that was shared with you (it runs right in your phone or
  computer browser — no install, no sign-up).
- Your data is saved **only in your own browser** on your own device. Nothing is
  shared, and you won't see anyone else's data. Clearing your browser data resets it.
- To begin: go to the **Settings** tab → type a name → **Create** a profile.
  Then add a few exercises (**Schedule** tab → **＋ Add exercise**).
- There are four tabs at the bottom: **Today · Schedule · Progress · Settings**.

If anything looks wrong or confusing, note **what you did** and **what happened** —
that's the most helpful feedback.

---

## 1. Adding several exercises keeps the same day
**What it does:** When adding exercises one after another, it remembers the day you
last picked instead of jumping back to today each time.

**Try it:**
1. **Schedule** tab → **＋ Add exercise**.
2. Change the **Day** to e.g. **Saturday**, fill in the rest, tap **Save**.
3. Tap **＋ Add exercise** again — the **Day** should still be **Saturday**.
4. Add one more — still **Saturday**. (So you can load up a day quickly.)
5. **Refresh the page**, then **＋ Add exercise** — now it defaults back to **today**.

---

## 2. Reps and weight *per set*
**What it does:** Each exercise can have different reps and an optional weight for
each set — not just one number for all sets. You set a default, and it fills every
set; then you can tweak any individual set.

**Try it:**
1. **Schedule** → **＋ Add exercise**. Enter a name.
2. Set **Sets** to **3**, **Default reps** to **10** → the 3 set rows below all show 10.
3. Type a **Default weight** of **20** → every set's weight fills with 20.
4. Now change one set's reps (e.g. set 3 → **15**) and one set's weight (e.g. → **25**).
5. **Save.** The card should summarise it sensibly, e.g. **3 × 10,15** and **weights vary**.
6. Change **Sets** to **5** in the form → two new rows appear (pre-filled from the
   defaults). Drop it to **2** → rows shrink. Save and check the card.

## 3. Workout mode that saves as you go
**What it does:** When you train, you can edit each set's reps/weight, tick sets
off, and even add or remove sets **just for that day** — and it **saves
automatically** as you go. Your saved exercise (the plan) never changes.

**Try it:**
1. Make sure today has at least one exercise planned (e.g. **Pull Ups, 3 × 10**,
   weight 20). The **Today** tab will show a **▶ Start workout** button.
2. Tap **Start workout.** Each exercise shows a row per set with the **reps** and
   **weight** filled in from the plan, plus a circle to tick the set **done**.
3. Tick a set (it turns green ✓ and the count goes up). Change a set's reps or weight.
4. Tap **＋ Add set** to add an extra set; tap the **✕** on a row to remove one.
5. **Don't finish yet** — tap **Close** (top-right). The button on Today should now
   say **▶ Resume workout**.
6. **Refresh the page**, tap **Resume workout** — everything you changed is still there.
7. Check the **Schedule** tab — the exercise still shows its original plan (your
   on-the-day changes didn't change it). ✅
8. Go back, tap **Resume** → **Finish**. You'll get a "Workout saved" message and it
   appears under **Recent workouts** on the Today tab.
9. (Optional) Start another workout and tap **Discard** instead — it deletes that
   day's progress without saving.

**Rest timer:** while in a workout, tap **60s / 90s / 120s** to start a countdown
(it beeps when done — make sure your phone isn't on silent).

---

## 4. Edit or delete a past workout
**What it does:** From your workout history you can fix a workout you already saved,
or remove one entirely.

**Try it:**
1. On the **Today** tab, find a finished workout under **Recent workouts**.
2. Tap **Edit** — it reopens that workout. Change a rep count, a weight, or tick/untick
   a set, then tap **Finish**. Your change is saved (the workout keeps its original date).
3. Tap **Delete** on a workout → confirm → it's removed from history, and the
   **Progress** tab updates to match.
4. **Refresh** to confirm your edits/deletions stuck.

## Also worth a look
- **Progress tab:** a "This week" summary and a small chart per exercise. **Tap a bar**
  to see that workout's details.
- **Dark mode:** the 🌙 button (top-right) switches light/dark — the sun mascot
  becomes a moon.
- **Backup (Settings tab):** **Export** saves your data to a file; **Import** loads it
  back (handy if you want to move data between browsers).
